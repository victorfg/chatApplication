# chatApplication
A Real Time Chat Application built using Node.js, Express, Handlebars, Mongoose, Socket.io

## Running the prototype

* `git clone` this repo
* `cd chatApplication` and `npm install`
* run `npm start`
* open browser on http://localhost:5000/
